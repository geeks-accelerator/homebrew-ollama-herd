# Homebrew Tap for Ollama Herd

Smart multimodal router for Ollama — LLM inference, image generation, speech-to-text, and embeddings across your device fleet.

## Install

```bash
brew tap geeks-accelerator/ollama-herd
brew install ollama-herd
```

## Usage

```bash
# Start the router
herd

# Start a node agent on each device
herd-node
```

Dashboard: http://localhost:11435/dashboard
